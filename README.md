## Purpose
Develop a web scraping pipeline to collect data from at least five web pages, clean it, and store it in a Pandas DataFrame. Perform and visualize data analysis using Seaborn to answer at least four questions, including two shown through labeled and styled plots.

## Analysis Questions Answered
1. Who were the top 5 home run hitters in each MLB season?
2. Which teams had the highest combined OPS each year?
3. Which team has more than 100 wins for two consecutive years or more?
4. Who were the top 5 pitchers with more strikeouts in each MLB season?

## Data sources
All raw data for team statistics, batting leaders, and pitching leaders was scraped from **Baseball-Reference.com**.

The specific pages targeted for analysis were the league summary pages for each season:
* `https://www.baseball-reference.com/leagues/majors/2024.shtml`
* `https://www.baseball-reference.com/leagues/majors/2023.shtml`
* `https://www.baseball-reference.com/leagues/majors/2022.shtml`
* `https://www.baseball-reference.com/leagues/majors/2021.shtml`
* `https://www.baseball-reference.com/leagues/majors/2020.shtml`
